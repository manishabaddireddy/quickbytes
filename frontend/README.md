# QuickBytes React Food Delivery App

QuickBytes is a React + Vite frontend for a cloud food delivery and dining app.

## Features

- QuickBytes browser title and animated QB logo
- Welcome screen with Delivery and Dining choices
- Delivery menu with search, category filters, animated cards, lazy-loaded real food images, euro prices, ratings, and cart
- Dining menu with the same food menu plus 20-seat booking interface
- Left icon sidebar, sticky top navbar, login/register animated modal
- Footer with information, contact, app download buttons, and social links

## Run Locally

1. Install Node.js from https://nodejs.org
2. Open this folder in VS Code.
3. Open the VS Code terminal.
4. Run:

```bash
npm install
npm run dev
